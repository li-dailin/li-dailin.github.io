# Dailin Li's Homepage

Personal academic homepage built with [Jekyll](https://jekyllrb.com/) and the [Academic Pages](https://github.com/academicpages/academicpages.github.io) theme, hosted on GitHub Pages at [li-dailin.github.io](https://li-dailin.github.io).

## Running Locally

1. Install dependencies: `sudo apt install ruby-dev ruby-bundler nodejs`
2. Install Ruby gems: `bundle install`
3. Build and serve: `bundle exec jekyll serve -l -H localhost`
4. Preview at `http://localhost:4000`

## Project Structure

- `_pages/` — Site pages (about, blogs, research, information)
- `_posts/` — Blog posts
- `_publications/` — Publication entries
- `_data/navigation.yml` — Navigation links
- `_config.yml` — Site-wide configuration
- `images/` — Image assets
- `files/` — Supplementary files (visualizations, archives)
